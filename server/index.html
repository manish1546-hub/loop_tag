Streamline your Express.js project setup and module creation!

About the Plugin

The Express Template CLI Plugin is a powerful command-line tool designed to simplify the creation of Express.js projects and modules. It automatically sets up directories, files, and configurations, including essential packages like mongoose, express, and express-async-handler, as well as development tools like nodemon.

Features

  • Generate a basic Express.js project structure with a single command.
  • Create modules with associated controllers, routes, and optional models.
  • Automatically installs required dependencies and development tools.
  • Adds support for nodemon for automatic server restarts during development.

Installation

Install the plugin globally using npm:

npm install -g express-template-cli

Commands

Below are the available commands for the plugin:

  • Generate a Basic Project: 
    create-express-template express-basic-template

    Sets up a new Express.js project with directories for controllers, routes, and models. Also creates a server.js file if it doesn't exist.

    Command Illustration

    config

    dbConfig.js

    controllers

    models

    routes

    server.js

  • Create a Module: 
    create-express-template express-module <module-name> [--model-none]
    Creates a new module with:
    • controller file
    • route file
    • model file (unless --model-none is passed)
    Example: 
    create-express-template express-module admin

    Command Illustration

    config

    dbConfig.js

    controllers

    adminController.js

    models

    adminModel.js

    routes

    adminRoutes.js

    server.js

  • Create Multiple Modules: 
    create-express-template express-module <module-name1> <module-name2> <module-name3> [--model-none] (--model-none is for all the modules passed if skipped model is created to all modules)
    Creates modules with:
    • controller file
    • route file
    • model file (unless --model-none is passed)
    Example: 
    create-express-template express-module admin products blogs

    Command Illustration

    config

    dbConfig.js

    controllers

    adminController.js

    productsController.js

    blogsController.js

    models

    adminModel.js

    productsModel.js

    blogsModel.js

    routes

    adminRoutes.js

    productsRoutes.js

    blogsRoutes.js

    server.js

  • Create Simple User System: 
    create-express-template express-simple-user [--jwt-token-none]
    Creates a new module with:
    • controller UserController.js file with functions
      • loginUser
      • registerUser
      • currentUser (unless --jwt-token-none is passed)
    • middleware AuthMiddleware.js file (unless --jwt-token-none is passed)
    • model UserModels.js file with the fields
      • Email
      • Username
      • Password
      • Phone
    • route UserRoutes.js file with routes
      • /login (method post)
      • /register (method post)
      • /current (method get) (unless --jwt-token-none is passed)
    Example: 
    create-express-template express-simple-user

    Command Illustration

    config

    dbConfig.js

    controllers

    userController.js

    models

    userModel.js

    routes

    userRoutes.js

    server.js
